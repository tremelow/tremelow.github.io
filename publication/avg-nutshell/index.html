Averaging in a Nutshell

Philippe Chartier, Mohammed Lemou, Florian Méhats, Léopold Trémant
January 2022
PDF

Abstract

Summary of some recent results concerning averaging, particularly stroboscopic averaging. We provide some original proofs of the geometric properties (volume-preservation, Hamiltonian…) of the average vector field, using only it definition. This differs from the historical approach which requires a construction of the said field.
